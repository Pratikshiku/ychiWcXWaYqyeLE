## Smart Community Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/75de53bc4efb40a7b9ad6ac35380e136/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular Users, Administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Supermarket Products
5. Animal Information
6. Parking Space Information
7. Community Services
8. Property Information
9. Property Staff
10. Community Announcements
11. Backend Management
12. Personal Center
13. My Favorites


**Administrator Functions:**
1. Login
2. System Homepage (Animal Type Statistics, Parking Space Statistics, Sales Statistics)
3. Personal Center (Change Password, Personal Information)
4. User Management
5. Product Category Management
6. Supermarket Product Management
7. Animal Type Management
8. Animal Information Management
9. Parking Space Information Management
10. Service Type Management
11. Community Service Management
12. Payment Information Management
13. Property Type Management
14. Property Information Management
15. Property Staff Management
16. Order Information Management
17. Feedback Management
18. System Management (System Introduction, Banner Management, Community Announcements, About Us)
